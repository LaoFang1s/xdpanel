系统介绍：

支持多种转发方式

流量收集重做 区分上下行

新增插件管理模式

新增一键救援模式

多服务器、多用户、多端口管理

自动化达量停机、超时停机

轻量平台式管理，中转机无被控，无机型要求 nat、vps、vds、杜甫均可使用

优化操作体验，UI友好 支持移动端操作
